# 🚀 Push to GitHub Instructions

## Option 1: Use Git Bash (Recommended)
1. Open Git Bash (usually installed with Git for Windows)
2. Navigate to your project:
   ```bash
   cd /c/baraka
   ```
3. Run these commands:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - Baraka DevX cybersecurity platform"
   git branch -M main
   git remote add origin https://github.com/barakadevx-afk/myportifolio.git
   git push -u origin main
   ```

## Option 2: Use GitHub Desktop
1. Download and install GitHub Desktop
2. Open GitHub Desktop
3. File → Add Local Repository
4. Select the `c:/baraka` folder
5. Commit changes with message "Initial commit"
6. Publish repository to GitHub
7. Repository URL: https://github.com/barakadevx-afk/myportifolio.git

## Option 3: Install Git to PATH
1. Download Git from: https://git-scm.com/download/win
2. During installation, ensure "Use Git from the Windows Command Prompt" is selected
3. Restart your computer
4. Open Command Prompt and run:
   ```cmd
   cd c:\baraka
   git init
   git add .
   git commit -m "Initial commit - Baraka DevX cybersecurity platform"
   git branch -M main
   git remote add origin https://github.com/barakadevx-afk/myportifolio.git
   git push -u origin main
   ```

## Option 4: Use VS Code
1. Open VS Code
2. File → Open Folder → Select `c:/baraka`
3. Open terminal (Ctrl + `)
4. Run the Git commands from Option 1
